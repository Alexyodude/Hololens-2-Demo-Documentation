    Create Box

    This script creates boxes and destroys them once they leave the set boundary.

    The first import is for the List class and the second import is for the classes and methods used in unity such as Rigidbody and MonoBehaviour. MonoBehaviour contains lifecycle methods such as Update() used in this Script. For more detail look into Unity - Scripting API: MonoBehaviour (unity3d.com).

    Public variables are declared to take in inputs from the unity scene and assets. A private list contains a list of box objects that were instantiated.

    The method "public void MakeBox()" instantiates and adds the object to the private list. In each Update() cycle the list of boxes are checked if it left the boundary that was set. If it does then it calls the Destory() method and removes the object that left the boundary from the list.

    The Prefab contains a rigidbody and object manipulator component.

    using System.Collections.Generic; // Import the System.Collections.Generic namespace for using the List class
using UnityEngine; // Import the UnityEngine namespace to access Unity-specific classes and methods

public class CreateBox : MonoBehaviour {

    // Public variables allow for easy assignment in the Unity Inspector
    public Rigidbody box; // Reference to the Rigidbody component that will be instantiated as a box. A prefab object was used.
    public GameObject location; // Reference to the GameObject representing the location where boxes will be instantiated
    private List<Rigidbody> list = new List<Rigidbody>(); // Private list to keep track of instantiated Rigidbody boxes

    // Method to instantiate a box and add it to the list
    public void MakeBox()
    {
        // Instantiate a new box at the location's position with the object's rotation
        Rigidbody newBox = Instantiate(box, location.transform.position, transform.rotation);
        // Add the new box to the list for tracking
        list.Add(newBox);
    }

    // Update is called once per frame
    void Update()
    {
        // Use a temporary list to store items to be removed to avoid modifying the list during iteration
        List<Rigidbody> itemsToRemove = new List<Rigidbody>();

        // Iterate over the list of boxes
        foreach (var item in list)
        {
            // Check if the box's y position is below -10
            if (item.transform.position.y < -10)
            {
                // Mark the box for removal
                itemsToRemove.Add(item);
            }
        }

        // Remove the marked boxes from the list and destroy their game objects
        foreach (var item in itemsToRemove)
        {
            // Destroy the game object associated with the Rigidbody
            Destroy(item.gameObject);
            // Remove the Rigidbody from the list
            list.Remove(item);
        }
    }
}
